Règles du Jeu


Pour jouer au Jeu des Fonctions, voici quelques règles importantes :

  • Les espaces et majuscules ne sont pas pris en compte dans la réponse.
    Par exemple, sin(x) est équivalent à SIN (X )

  • Utilisez la syntaxe informatique pour les fonctions. Voici quelques exemples :
x*2 pour la multiplication.
x/2 pour la division.
x^y pour x puissance y.
sqrt(x) pour la racine carrée.
sin(x) pour le sinus.
cos(x) pour le cosinus.
tan(x) pour la tangente.
exp(x) pour l'exponentielle.
ln(x) pour le logarithme.
abs(x) pour la valeur absolue.

Les réponses peuvent être données dans n'importe quel ordre en respectant les
règles mathématiques de base, ainsi : x*5 = 5*x mais : 1/x ≠ x/1

Mettez vos compétences à l'épreuve et amusez-vous à retrouver les fonctions !
